# carolinasoares.dev

Blog desenvolvido com [Hugo](https://gohugo.io/) e com o tema [Gokarna](https://github.com/gokarna-theme/gokarna-hugo).

- Baixar tema:

      git submodule add https://github.com/gokarna-theme/gokarna-hugo.git themes/gokarna

- Compilar:

      hugo

- Executar:

      hugo server
